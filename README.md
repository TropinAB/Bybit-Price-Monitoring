# Bybit Crypto Monitor

Приложение для мониторинга цен криптовалют на бирже Bybit с возможностью установки целевых цен и получения уведомлений.

## 📋 О проекте

Приложение предназначено для отслеживания цен криптовалют на бирже Bybit и мониторинга установленных целевых цен. Вы можете добавлять инструменты в список наблюдения, устанавливать целевые цены и получать уведомления при их достижении.

### Основные возможности

- 📈 **Просмотр инструментов** - таблица со всеми доступными инструментами Bybit с сортировкой и фильтрацией
- 🔍 **Детальная информация** - просмотр подробной информации о выбранном инструменте
- 🎯 **Установка целевых цен** - добавление инструментов в мониторинг с указанием желаемой цены
- 🔔 **Уведомления** - всплывающие toast-уведомления и звуковые сигналы при достижении цели
- 📋 **Мониторинг целей** - отдельная страница со списком отслеживаемых инструментов
- 💾 **Сохранение данных** - автоматическое сохранение всех настроек в localStorage

## 🚀 Технологии

- **React 19** - современная библиотека для пользовательских интерфейсов
- **TypeScript** - типизация для надёжного кода
- **CSS Modules** - стилизация компонентов
- **Bybit API** - получение данных с биржи
- **LocalStorage** - хранение пользовательских настроек
- **Jest** - тестирование

## 📦 Установка и запуск

### Требования

- Node.js 18+
- npm или yarn

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/TropinAB/Bybit-Price-Monitoring.git

# Перейти в папку проекта
cd Bybit-Price-Monitoring

# Установить зависимости
npm install
# или
yarn install
```

### Запуск

```bash
# Режим разработки
npm run dev
# или
yarn dev

# Сборка для продакшна
npm run build
# или
yarn build

# Запуск тестов
npm test
# или
yarn test
```

Приложение будет доступно по адресу `http://localhost:3000`

## 📖 Как пользоваться

1. **Инструменты** - на главной странице отображается таблица всех доступных инструментов Bybit. Используйте фильтры для выбора категории и базовой валюты, сортируйте данные по любому столбцу.

2. **Установка цели** - нажмите кнопку **(+)** рядом с инструментом и введите желаемую цену. Инструмент будет добавлен в мониторинг.

3. **Мониторинг** - на странице мониторинга отслеживайте все установленные цели. При достижении цели вы получите уведомление и звуковой сигнал, а строка подсветится зелёным.

4. **Управление** - удаляйте отдельные цели или очищайте весь список кнопкой "Очистить все данные".

## ⚙️ Настройки

### Интервал обновления

Частота обновления цен настраивается от 0 до 5 минут. Выбранное значение сохраняется в localStorage.

### Фильтры

- **Тип инструмента** - спот, бессрочные контракты (PERP) или все
- **Базовая валюта** - фильтр по базовой монете (BTC, ETH и т.д.)

### Сортировка

Клик по заголовку колонки сортирует данные:

- ↑ - по возрастанию
- ↓ - по убыванию

## 🔔 Уведомления

При достижении целевой цены:

- Появляется toast-уведомление в правом верхнем углу
- Воспроизводится звуковой сигнал (колокольчик)
- Строка в таблице мониторинга подсвечивается зелёным

## 💾 Хранение данных

В localStorage сохраняются:

- Список целей для мониторинга
- Настройки частоты обновления
- Выбранная категория фильтра
- Выбранная базовая монета
